## Simple Python Sudoku App

![Demo gif](demo.gif)

## Installation
1. Install the [`carbon`](https://github.com/nvfp/carbon) module
2. Clone/download this repository and save it to your machine (e.g. ~/projects/Sudoku)
3. Run it:
    ```sh
    python Sudoku
    ```
